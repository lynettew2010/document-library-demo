About the site

Built to be straightforward, readable, and low-maintenance.

This layout is intentionally simple: a homepage, category sections, a document library, and optional HTML pages for content that should be read directly in the browser instead of downloaded.

Easy long-term update options:

  • Add new PDFs manually to a folder and link them in the HTML
  • Duplicate a document card for each new file
  • Convert important PDFs into separate HTML pages later
